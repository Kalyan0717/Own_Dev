										Sup, Sub, Strong And em, Special Chars
										---------------------------------------
Sup -> Means Separate (Content showing in word upper)
Sub -> Means content showing down the line word
Strong -> means content showing the bold letters 
em -> Means Empass 
Special Characters -> 
					For Example:
					------------
					&copyright
InPut:
-------
	<!DOCTYPE html>
	<html lang="en">
	<head>
		<meta charset="UTF-8">
		<meta name="viewport" content="width=device-width, initial-scale=1.0">
		<title>Sup, Sub And Strong</title>
	</head>
	<body>
		<h1>Hii im the<sup>1st</sup> Kalyan <sub>in my town</sub></h1>
		<p>Lorem ipsum dolor sit amet consectetur adipisicing elit. 
			<strong>Dolore earum eveniet rem ipsa, cupiditate, consequatur illum voluptatum perspiciatis id 
			</strong>in minima, deleniti quo nesciunt ab dicta.
			<em>Fuga veritatis a magni architecto possimus debitis quam odit,</em>  
			dolor error eaque atque facere perspiciatis corrupti laborum tenetur id dolores sequi dolorem ipsa magnam.</p>
			<h2>&dagger;<br>&ac;<br>&longmapsto;</h2>
	</body>
	</html>

														Lists
										Un - Orderd List, Orderd List, Nested List
										------------------------------------------
1) Un - Orderd List:
--------------------
	->	<ul></ul> -> Syntax of the Un-Orderd it is also known as ul-tag
		<li> -> Syntaxof the List (list tag)
		
	-> Un-Orderd List Not in the Numaric numbers 
	-> Un-Orderd used in the anchor tags (some of our recuriment)
			
			<!-- Un-Orderd list -->
			<h1>Un-Orderd list</h1>
			<ul>
				<li><a href="#">Home</a></li>
				<li><a href="./AboutPage.html">About</a></li>
				<li><a href="#">Contact</a></li>
			</ul>
	
2) Orderd List:
---------------
	->	<ol></ol> -> Syntax of the Orderd it is also known as ol-tag
		<li> -> Syntaxof the List (list tag)
		
	-> Orderd List in the Numaric numbers 
	-> Orderd used in the anchor tags (some of our recuriment)
	
			<!-- Orderd list -->
			<h1>Orderd list</h1>
			<ol>
				<li>Kalyan</li>
				<li>Kittu</li>
				<li>Pavan</li>
			</ol>
	
3) Nested List:
---------------
	-> Nested List having both Un-Orderd And Orderd is known as Nested List
		
			<!-- Nested list -->
			<h1>Nested list</h1>
			<ul>
				<li>Kalyan</li>
				<li>Kittu</li>
				<ol>
					<li>HTML</li>
					<li>CSS</li>
				</ol>
				<li>Pavan</li>
			</ul>
